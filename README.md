# 📚 Solidity Mastery: Curso de Smart Contracts

Bienvenido al repositorio con los conceptos fundamentales y avanzados para aprender a programar **Smart Contracts en Solidity**. Este material ha sido diseñado para llevarte desde las bases del lenguaje hasta la arquitectura de sistemas descentralizados complejos.

---

## 🏛️ Institutional Note
> This repository is an official resource developed for the **Blockchain Laboratory** at the **National Technological University – La Plata Regional Faculty (UTN FRLP)**. It is designed to provide students with a high-level technical foundation in Smart Contract engineering and Web3 development.

---

## 📖 Cómo utilizar este curso

Para aprovechar al máximo el contenido, te recomendamos seguir este flujo de aprendizaje en cada unidad:

1.  **Teoría (`.md`):** Comenzá leyendo el archivo Markdown. Aquí encontrarás la explicación conceptual, los estándares involucrados y la justificación técnica de cada tema.
2.  **Práctica (`.sol`):** Una vez entendida la teoría, revisá el código del contrato de ejemplo. Analizá cómo se implementan los conceptos y qué librerías de OpenZeppelin se utilizan.
3.  **Ejercicios (`ejercicios.sol`):** Abrí este archivo en tu editor (Remix, VS Code) y tratá de resolver los desafíos planteados. Es la parte más importante para fijar el conocimiento.
4.  **Soluciones (`ejercicios-resueltos.sol`):** Si te trabás o querés comparar tu enfoque con las mejores prácticas de seguridad, consultá el archivo de soluciones.
